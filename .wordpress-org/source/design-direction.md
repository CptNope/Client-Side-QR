# Asset Direction

## Positioning

Client-Side QR Code Generator should read as:

- privacy-friendly
- practical for WordPress publishing and campaigns
- modern, but not flashy
- trustworthy enough for business, agency, and site-builder use

## Visual direction

- Primary palette:
  - Ink: `#17313b`
  - Teal: `#2a7f87`
  - Mint: `#9ed6c2`
  - Sand: `#f5efe4`
  - White: `#ffffff`
- Style:
  - soft editorial background
  - clean geometry
  - QR-inspired pattern blocks
  - rounded browser/device framing to suggest client-side rendering

## Banner copy

- Eyebrow: `Privacy-friendly WordPress QR workflows`
- Title: `Client-Side QR Code Generator`
- Supporting line: `Generate styled QR codes in the browser for links, campaigns, contact sharing, WiFi, and payments.`

## Icon direction

- Rounded square field
- High-contrast QR-inspired corner modules
- Center path mark suggesting flow, publishing, and generation
- Readable at small sizes without relying on fine detail
